---
title: "Website Categorization"
api: "GET https://categorization.dev/api/v1/website/categorization"
description: "This endpoint allows for categorizing websites based on NAICS (North American Industry Classification System) codes or a taxonomy classification."
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    This parameter specifies the private key you'll need for Categorization.dev access.
</ParamField>

### Parameters

#### Search Query

<ParamField query="query" type="string" required={true} default="https://airbus.com">
    A website url with `http://` or `https://` protocol.
</ParamField>

```bash Taxonomy v3 Example Request
curl --location --request GET 'https://categorization.dev/api/v1/website/categorization' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
--data-urlencode 'classification=taxonomy_v3' \
--data-urlencode 'query=https://airbus.com'
```

```bash NAICS Example Request
curl --location --request GET 'https://categorization.dev/api/v1/website/categorization' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
--data-urlencode 'classification=naics' \
--data-urlencode 'query=https://airbus.com'
```

</RequestExample>
